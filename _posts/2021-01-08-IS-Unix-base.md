# 유닉스 발전

![Unix_tree](/assets/images/posts/unix_tree.jpg)

[이미지 출처](https://slidesplayer.org/slide/11036403/)

유닉스는 크게 두 계통으로 나뉨
- 시스템5 : 상업용
- BSD : 연구개발용

유닉스의 발전
- 유닉스 : AT&T, 벨 연구소
- 버전 3 : 데니스리치 (C언어)
- 버전 6 : 외부 공개
- 1.x BSD : 버클리 대학에서 네트워크 기능을 추가
- 버전 7 : 본쉘 등장, 유닉스의 원조
- SunOS 1.x : Sun 사에서 BSD 를 토대로 개발
- 시스템3 : AT&T 사에서 개발
- SunOS 2.x : BSD + 시스템5

유닉스 종류
- IBM AIX (시스템5)
- HP-UX (시스템5)
- Solaris (시스템5)
- SunOS (BSD)
- FreeBSD (BSD)
- 리눅스
